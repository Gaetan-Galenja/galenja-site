# Galenja Website (GitHub Pages)

Static site for Galenja (Automation Microsoft 365).

## Local preview
Open `index.html` in a browser.

## Deploy on GitHub Pages
- Push these files to your repository (e.g., `galenja-site`).
- In GitHub → **Settings → Pages**:
  - **Source**: `Deploy from a branch`
  - **Branch**: `main` and **Folder**: `/ (root)`
- Custom domain: set to `galenja.fr` (a `CNAME` file is included)
